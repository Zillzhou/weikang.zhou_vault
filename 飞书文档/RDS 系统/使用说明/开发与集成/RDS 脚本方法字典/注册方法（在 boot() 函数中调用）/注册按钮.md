# 注册按钮

## 注册按钮
## 方法说明
向rds注册管理后台可见的按钮。
function registerButton(label: string, remark: string, scriptFunction: string, level: string): void

## 输入参数
label，string 类型，按钮名。
remark，string 类型，按钮备注。
scriptFunction，string 类型，点击按钮触发的方法名， 注意该方法不能带参数 。
level，string 类型，按钮等级，用颜色区分。blue：蓝色，yellow：黄色 ，red：红色。
## 输出参数
## 无
## 异常
本方法会抛出异常。
