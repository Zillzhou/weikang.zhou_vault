# Overview

## Overview
本手册是 RDS 的API 使用手册，旨在帮助用户通过 API 接入相关数据到内部系统，从而实现获取相关数据的目的。
## 基本路径
### [ Base URL: host:8080/ ]
例如服务器IP为 192.168.1.100 的程序，API的基础路径为 http://192.168.1.100:8080/
接口的请求正文和响应正文都是 json 格式的数据，编码格式为 UTF-8 。
