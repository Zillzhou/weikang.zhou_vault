# 修改单个库位占用状态

## 修改单个库位占用状态
## 接口介绍
此 API 可以修改一个指定库位的状态。
## 请求
## 功能：修改一个指定库位的状态
## 方法： POST
接口说明： /api/work-sites/updateFilledStatus
POST "http://host:8080/api/work-sites/updateFilledStatus"
## 请求参数

## Name

## Type

## Description

## Required

## siteId

## String

## 目标库位（工位）的编号

## 是

## filled

## int

## 库位占用状态（0：未占用，1：占用）

## 是
## 请求示例
## {
    "siteId": "Loc-01",
## "filled": 1
## }
## 响应

## Name

## Type

## Description

## code

## int

## API 错误码，详情见 API 错误码

## msg

## String

## API 错误码信息
## 响应数据示例
## Responses Code 200
## {
    "code": 200,
## "msg": "Success"
## }
