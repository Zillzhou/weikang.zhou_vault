# 撤销/终止任务

## 撤销/终止任务
## 接口介绍
此 API 可以撤销/终止任务，选择是否解锁任务相关的库位。
## 请求
功能：撤销/终止任务，选择是否解锁任务相关的库位。
## 方法：POST
### 接口说明： /api/stop-task
## 请求示例的 URL
POST http://host:8080/api/stop-task
## 请求参数

## Name

## Type

## Description

## Required

## releaseSite

## int

### 是否同时解锁任务相关的库位（0：否，1：是）

## 是

## taskRecordId

## String

## 终止的任务实例ID

### 是，和 taskLabel 字段 必须选填一个

## taskLabel

## String

## 终止的任务名称

是，和 taskRecordId 字段 必须选填一个 ，如果 填了  taskRecordId 此字段无效
## 请求示例
终止任务实例 ID 为 bd536d07-b089-4c50-b058-e7cdcfd3a194 的任务，并解锁任务相关的库位。
## {
  "releaseSite": 1,
  "taskRecordId": "bd536d07-b089-4c50-b058-e7cdcfd3a194"
## }
