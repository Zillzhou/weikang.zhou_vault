# 任务滞留列表

## 任务滞留列表
### 适用 rds-F1.9.5-B1.7.20 及以上
## 接口介绍
此 API 可以查询任务滞留列表。
## 请求
## 功能：查询任务滞留列表
## 方法：POST
接口说明： /api/stat/taskRetentionList
## 请求示例的 URL
POST http://host:8080/api/stat/taskRetentionList
## 请求参数

## Name

## Type

## Description

## Required

## currentPage

## int

### 表示查询第 currentPage 页的结果

## 是

## pageSize

## int

### 表示按照每页 pageSize 条记录，对所有结果进行分页

## 是

## queryParam

## Object

## 查询参数对象

## 是

## retentionHours

## String

## 滞留时间，单位：小时

## 是
## 请求示例
## 分页查询滞留时间超过1小时的任务列表
## {
    "currentPage": 1,
    "pageSize": 10,
## "queryParam": {
## "retentionHours": 1
## }
## }
## 响应

## Name

## Type

## Description

## code

## int

## API 错误码，详情见 API 错误码

## msg

## String

## API 错误码信息

## data

## Object

## 返回的数据对象

## totalCount

## int

## 表示查询到的任务的总数

## currentPage

## int

## 表示当前结果的页码

## pageSize

## int

## 表示当前页可记录的任务的最大数量

## totalPage

## int

### 表示查询到的所有任务，总共被分成了 totalPage 页

## pageList

## List

## 记录了若干条任务信息

## id

## String

## 表示当前任务实例的Id

## defLabel

## String

## 天风任务名

## createOn

## String

## 任务运行时间

## status

## int

任务状态（1000:运行中；1001:终止；1002：暂停；1003：结束；1004：异常结束）

## agvId

## String

## 运行的机器人Id
