# 机器人当日用电趋势

## 机器人当日用电趋势
### 适用 rds-F1.9.44-B1.7.57 及以上
查询机器人一天用电趋势。源数据：Core 获取所有机器人信息
统计周期：每分钟更新一次数据。
## 请求
## 方法： POST
接口说明： /api/stat/ vehicleBatteryLevelTrend
POST "http://127.0.0.1:8080/api/stat/vehicleBatteryLevelTrend"
## 请求body 参数

## Name

## Type

## Description

## vehicle

## string

## 机器人Id

## start

## string

## 查询日期
## 请求示例
## {
  "start": "2023-03-31",
## "vehicle": "AMB-02"
## }
## 响应
排除接口错误或网络异常等原因，响应的 HTTP 状态码为 200。响应体中有不同业务码，成功为 200，失败业务码可参考 业务错误码列表 。
## 响应 body 参数

## Name

## Type

## Description

## code

## number

## 业务码(非 HTTP 状态码)

## msg

## string

## 业务成功/失败信息

## data

## object

## 请求的业务数据

## vehicleId

## string

## 机器人Id

## batteryLevel

## float

## 机器人电量

## time

## string

## 电量记录时间
## 响应示例
## {
    "code": 200,
    "msg": "Success",
## "data": [
## {
            "vehicleId": "AMB-02",
            "batteryLevel": 1.00,
### "time": "2023-03-31 17:00:00"
        },
## {

            "batteryLevel": 1.00,
### "time": "2023-03-31 17:05:00"
        },
## {
            "vehicleId": "AMB-02",
            "batteryLevel": 1.00,
### "time": "2023-03-31 17:10:00"

响应数据中"time"为每五分钟整记一次。未到五分钟，时间记录为下一次五分钟，每分钟更新一次数据。电量数据只保留七天。
