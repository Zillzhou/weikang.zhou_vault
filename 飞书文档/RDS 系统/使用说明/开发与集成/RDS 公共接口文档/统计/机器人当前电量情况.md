# 机器人当前电量情况

## 机器人当前电量情况
### 适用 rds-F1.9.44-B1.7.57 及以上
查询当前时间机器人电量情况。源数据：Core 获取所有机器人信息
统计周期：每分钟更新一次数据。

## 请求
## 方法： POST
接口说明： /api/stat/vehicleBatteryLevel
POST "http://127.0.0.1:8080/api/stat/vehicleBatteryLevel"
## 请求示例
## {
## }
## 此接口无参数
## 响应
排除接口错误或网络异常等原因，响应的 HTTP 状态码为 200。响应体中有不同业务码，成功为 200，失败业务码可参考 业务错误码列表 。
## 响应 body 参数

## Name

## Type

## Description

## code

## number

## 业务码(非 HTTP 状态码)

## msg

## string

## 业务成功/失败信息

## data

## object

## 请求的业务数据

## vehicleId

## string

## 机器人Id

## batteryLevel

## float

## 机器人电量

## time

## string

## 电量记录时间
