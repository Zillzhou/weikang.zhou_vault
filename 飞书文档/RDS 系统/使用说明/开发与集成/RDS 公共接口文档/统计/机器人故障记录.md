# 机器人故障记录

## 机器人故障记录
## 接口介绍
此 API 可以查询机器人故障记录。
## 请求
## 功能：查询机器人故障记录
## 方法：POST
### 接口说明： /api/stat/alarmsRecords
## 请求示例的 URL
POST http://host:8080/api/stat/alarmsRecords
## 请求参数

## Name

## Type

## Description

## Required

## currentPage

## int

### 表示查询第 currentPage 页的结果

## 是

## pageSize

## int

### 表示按照每页 pageSize 条记录，对所有结果进行分页

## 是

## queryParam

## Object

## 查询参数对象

## 是

## startCreateTime

## String

故障开始时间的开始（与endCreateTime，必须两个都填或两个都不填）

## 否

## endCreateTime

## String

故障开始时间的结束（与startCreateTime，必须两个都填或两个都不填）

## 否

## startEndTime

## String

故障结束时间的开始（与endEndTime，必须两个都填或两个都不填）

## 否

## endEndTime

## String

故障结束时间的结束（与startEndTime，必须两个都填或两个都不填）

## 否

## vehicleId

## String

## 机器人Id

## 否

## level

## String

### 故障等级(errors:错误；warnings:告警)

## 否
## 请求示例
分页查询机器人为sim_01，故障级别为errors，故障开始时间在2023-09-04 00 : 00 : 00 - 2023-09-04 23 : 59 : 59 的故障列表
## {
    "currentPage": 1,
    "pageSize": 10,
## "queryParam": {
        "startCreateTime": "2023-09-04 00:00:00", 
              "endCreateTime": "2023-09-04 23:59:59",
              "vehicleId": "sim_01",
## "level":"errors"
## }
## }
## 响应

## Name

## Type

## Description

## code

## int

## API 错误码，详情见 API 错误码

## msg

## String

## API 错误码信息

## data

## Object

## 返回的数据对象

## totalCount

## int

## 表示查询到的任务的总数

## currentPage

## int

## 表示当前结果的页码

## pageSize

## int

## 表示当前页可记录的任务的最大数量

## totalPage

## int

### 表示查询到的所有任务，总共被分成了 totalPage 页

## pageList

## List

## 记录了若干条任务信息

## alarmsCode

## String

## 报警码

## vehicleId

## String

## 机器人Id

## level

## String

### 故障等级(errors:错误；warnings:告警)

## startedOn

## String

## 故障的开始时间

## endedOn

## String

## 故障的结束时间

## alarmsCostTime

## BigDecimal

## 故障耗时（单位：min，保留两位小数）
