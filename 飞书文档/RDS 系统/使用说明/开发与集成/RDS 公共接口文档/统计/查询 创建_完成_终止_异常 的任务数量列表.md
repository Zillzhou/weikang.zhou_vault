# 查询 创建/完成/终止/异常 的任务数量列表

### 查询 创建/完成/终止/异常 的任务数量列表
### 适用 rds-F1.9.5-B1.7.20 及以上
统计周期：level为Hour，每十分钟更新一次数据；level为Day，Month，Year，每十五分钟更新一次数据。
### 查询 创建/完成/终止/异常 的任务数量列表
## 接口介绍
此 API 可以查询 创建/完成/终止/异常 状态的任务数量列表，只包含任务数据，不包含机器人相关信息。
## 请求
### 功能：查询创建/完成/终止/异常 状态的任务数量列表
## 方法：POST
### 接口说明： /api/stat/taskList
## 请求示例的 URL
POST http://host:8080/api/stat/taskList
## 请求参数

## Name

## Type

## Description

## Required

## level

## String

统计等级，支持 4种 参数， Hour , Day , Month , Year 。

## 是

## types

## String

统计的类型，见下表，需要统计多个类型时用逗号','隔开（WindTaskCreatedNum: 任务创建数量,WindTaskFinishedNum: 任务完成数量,WindTaskStoppedNum：任务终止数量，WindTaskExceptionNum：任务异常数量）
（注意：WindTaskExceptionNum在 B1.7.57版本 才有该维度的统计 ）

## 是

## start

## String

统计的开始时间（包含），支持 4种 格式的时间参数。 yyyy-MM-dd HH , yyyy-MM-dd , yyyy-MM , yyyy ，分别对应时报，日报，月报和年报，和参数 level 相关。

## 是

## end

## String

统计的结束时间（不包含），支持 4种 格式的时间参数。 yyyy-MM-dd HH , yyyy-MM-dd , yyyy-MM , yyyy ，分别对应时报，日报，月报和年报，和参数 level 相关。

## 是
## 请求示例
查询 2019年（包含）- 2022 年（不包含）的任务终止数量
## {
    "level": "Year",
    "types":"WindTaskStoppedNum",
    "start":"2019",
## "end":"2022"
## }
