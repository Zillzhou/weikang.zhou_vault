# RDS 移动端 APP 界面说明

## RDS 移动端 APP 界面说明
## 1、登录
1678857790204-f429202c-979a-40f1-8e4a-edeea6f7695b.png

首先，点击登录界面的【服务器配置】，弹出服务器地址配置的弹框。
1678857989104-5937ace4-f870-4eba-87f1-c966197967d9.png

在服务器地址配置的弹框中，可以选择服务器网络连接协议，目前支持 http 或者 https。其中 http 协议对应的端口是 8080，https 协议对应的端口是 8090。
在【IP】和【端口号】两个输入框中分别输入手持端准备连接的服务器的 IP 地址和端口号，然后点击保存，返回登录页。
1678857999849-23d6ef54-ab8c-4fe1-8158-ca487c777008.png

返回登录页后，在其中输入用户名和密码，点击登录手持端。
1678858701797-4ad59320-8237-43d0-8dc2-9ae750a275de.png

### 2、发起任务
登录后默认打开【发起任务】页面，界面中显示 yml 配置文件中添加的手持端任务菜单列表。
任务菜单可点击，点击后，弹出菜单详情。在弹出框完成相关的业务参数输入或选择后，点击【确定】按钮对服务器发起下单请求。
1678858738368-53f55425-13ac-4379-91be-6c2cf915e015.png

点击发起任务界面的右上角，将跳到岗位配置页面，可配置当前手持端登录用户的岗位、工位、工号，如果任务设置过岗位工位，则在选择岗位工位后，发起任务页面会根据岗位呈现相应的任务。
1678858775467-2ba93438-9d7a-4823-9120-797a846891fa.png

### 3、需求单
在设置和脚本中进行配置以后，手持端会出现【需求单】页面。例如，线边员工在发起任务页面发起叫料需求，点击“仓库出货”按钮，选择需要物料的库位：线边库位1。

1678858842583-a205d573-901a-48b9-ac68-5383109ed1bd.png

特定岗位可以看到【发起任务】页面的叫料需求。呈现在【需求单】页面，需求信息包含叫料名称、描述和创建时间。
1678858869589-7db8d972-53e0-4bc2-ab7d-8af6f55d9bfd.png

点击需求单列表中的需求信息，可以选择给叫料库位提供物料的岗位，选择以后生成任务，实现需求单的叫料需求。

## 其中的按钮功能：
【可接单】【不接单占资源】【不接单不占资源】：设置机器人接单/占用资源状态。
【终止运单】：终止正在运行的运单，同时设置为可接单状态。
【抢占控制权】：RDS 抢占机器人控制权。
【释放控制权】：RDS 释放机器人控制权。
【继续】：机器人继续导航。
【暂停】：机器人暂停导航。
【复位故障】：清除所有错误信息，复位。
【取消命令】：清除所有导航路径。
【确认定位】：确认机器人定位。
【软急停】：清除所有错误信息，复位。
【取消软急停】：软触发的车的急停。
【放行】：放行机器人。
机器人搜索。点击界面上的搜索框，输入机器人名字，可根据机器人名字筛选。
1701934244545-d7bbd3cc-50e7-47d4-b09f-d22f99e6338f.png

库位界面中勾选一个库位，点击【操作】按钮，左侧弹出库位支持的功能。

### 11、设置
设置界面可设置登录用户的岗位、工位、工号、绑定机器人，可以设置刷新频率，设置语言（中文、英文、日本语、繁体）。还可以选择声音模式：响铃+震动，添加提示音文件。

### 12、外部链接
手持端可以嵌入外部链接，只需要在RDS界面设置外部网页的名称和url，即可在手持端显示，此处显示仙工智能官网。

在【设置】页面中的下方点击【退出登录】按钮，则用户退出。
