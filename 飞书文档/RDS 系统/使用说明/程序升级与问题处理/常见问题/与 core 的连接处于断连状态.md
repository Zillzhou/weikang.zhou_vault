# 与 core 的连接处于断连状态

## 与 core 的连接处于断连状态
现象： RDS 弹出与core的连接处于断连状态，或者机器人菜单页面无机器人信息。
## Core （调度）未启动
windows 打开命令提示窗→以管理员身份运行，输入命令 curl http://127.0.0.1:8088/ping ，出现 code：0 字样即为成功。否则，需要启动Core。 Ubuntu 打开终端运行此命令。验证结果同 `windows ` 。
