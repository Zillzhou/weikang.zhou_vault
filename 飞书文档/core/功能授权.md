# 功能授权

## 功能授权
## 介绍
### RDS 与 RDSCore 的功能激活分成三个部分：
rds_custom 代表低代码，即激活后可以自行编辑天风任务
## rds_operator 为手持端激活
## core 为 RDSCore 激活

## 激活步骤
在调度服务器中点击高级设置，点击【版本信息】，此时页面右侧 RDSCore 的状态将显示【未激活】，激活该条表示获得标准版的 RDSCore 和 RDS 的授权。您需要复制【机器码】，将【机器码】提交我司项目经理，索要授权文件。后点击【离线激活】按钮，使用授权文件激活 Core 的对应权限。或点击【在线激活】直接激活。（ 注：激活的过程中，Core 将会重启，属正常现象 ）。
