## Group

## string

## 空

## 任意字符串

机器人组名称，组内的机器人采用此策略，可缺省，同时设置 group 和 label 则捕获同时满足的机器人

## Label

## string

## 空

## 任意字符串

机器人标签名称，此标签的机器人采用此策略，可缺省，同时设置 group 和 label 则捕获同时满足的机器人

## Begin

## string

## 00:00

## 00:00--23:59

禁止停靠起始时间，注意冒号采用英文符号，如设置 9:30，则表示从上午九点半开始持续 duration 分钟内禁止停靠

## Duration

## int

## 0

## 正整数

禁止停靠持续时间，单位为分钟，只支持整数，如设置 30，则表示从 begin 开始持续 30min 禁止停靠

## 注意事项
### 模型文件使用时需要勾选 启用设备 才起作用
策略捕获的机器人为同时满足设置的 group 和 label 的机器人
## 可以点击＋同时设置多个策略
当多个策略的时间段存在重合时，会进行合并，比如设置机器人组 g1 的机器人 17:00--18:00 和 17:30--18:30 禁止停靠，则最后结果为 17:00--18:30 禁止停靠
策略的最小精度是分钟，故如果设置 9:00--10:00 禁止停靠，那么机器人的开始停靠时间为 10:01:00 + startParkTime(延时停靠参数)
