# 自动任务

## 自动任务
## @丁禹伯
自动任务是指触发某些特定逻辑后，调度系统会自动为机器人发送订单，而不需要通过外部来进行发单；目前包含的自动任务类型有：
### 自动停靠 ：机器人无任务时自动返回停靠点
### 自动充电 ：机器人低电量时自动前往充电桩
火警 ：发生火警时，机器人原地不动 或 移动至安全区域停止不动，等待火警结束继续工作
## 预停靠： 带有优先级的停靠任务
### 上下班：通过触发下班状态来驱动机器人前往指定区域
目前自动任务分为新旧两种架构，两种架构主要是在底层实现上存在差异，在用户层上，两者均支持 充电、停靠、火警 与 预停靠，区别在于 新架构支持轮换充电、队列停靠、上下班，旧架构不支持轮换充电、队列停靠，并且处于维护简易的考虑，后续新的功能会倾向于在新架构上实现，逐渐抛弃旧架构

### 0.1.9.240911 版本开始，自动任务默认使用新架构，不再维护旧架构
