# MAPF 组

## MAPF 组
## @丁禹伯
Roboshop 2.4.1.95 版本后，左侧栏 互斥组 更名为 高级组
MAPF 组是一个图元的组合，包含线路，点、区域。在一个 MAPF 组区域内，可以设定能够同时进入该区域的机器人数量上限，MAPF 组可以看做是一个可以指定数量上限的互斥区。互斥区也等价于机器人数量上限为1的 MAPF 组。
MAPF 组中的除了限制最多机器人数量外，其他属性也会逐渐增加，用来调整MAPF区域内多机器人的路径导航。
### 最新功能可在这个文档里跟踪 区域属性配置（MAPF）
添加方式和互斥组一样，不过需要将这个互斥组的 type 选择成 MAPF，然后的将 maxRobotNum 属性配置成合适的机器人数量。

## Error52201
## 提示：
The number of robots in same block group is over max number.
## 触发条件：
机器人在MAPF区域中数量超限。
## 解决方式：
将互斥组中的机器人从互斥组中移开，或者机器人设置为不占用资源状态。
