# Priority 组

## Priority 组
## @黄强盛
Priority 组可包含点位，线路和区块。Priority 组可提高区域内机器人执行的优先级，当多个机器人抢占同一资源时，Priority 组内的机器人将具有更高的优先级，可以优先占用互斥资源。
 在实际应用中，Priority 组的设置可以根据具体需求进行灵活调整。例如，可以将重要的点位、线路或区块设置为 Priority 组，以确保调度能够优先处理这些关键区域的机器人，让机器人优先抢占资源。同时，也可以根据任务的紧急程度和重要性，对 Priority 组的优先级进行动态调整，优先级越大，机器人越先抢到互斥资源
