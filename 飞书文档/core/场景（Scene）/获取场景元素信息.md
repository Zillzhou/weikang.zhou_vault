# 获取场景元素信息

## 获取场景元素信息
## 用于支持客户进行二次开发

## 请求

### POST ip:port/queryTopoInfo
## 请求JSON字段

## 名称

## 必填

## 描述

## robotGroup

## 是

## 机器人组

## topoElement

## 是

## 元素名称

## elementType

## 是

## 元素类型
## 元素类型
### 目前支持两种元素类型：point,edge
elementType为point时，对应的topoElement为点位名称；为edge时，topoElement为边的名称
## 元素名称
## 点位：LM123
线路：LM123-AP456表示从LM123到AP456的单向线路
## 响应
请求成功时，响应中的info字段含有对应的场景元素信息。失败时，error信息含有错误描述。
## 点位信息

## 名称

## 类型

## 含义

## name

## string

## 点位名称

## dir

## double

## 点位的朝向，单位rad

## ignore_dir

## bool

## 是否忽略点位朝向
## 路线信息

## 名称

## 类型

## 含义

## end

## int

## 线路终点

## start

## int

## 线路起点

## length

## double

## 线路长度，单位m

## max_speed

## double

### 线路上配置的最大速度，单位m/s，为负数时表示未设置

## start_dir

## double

## 线路起点处的朝向，单位rad

## end_dir

## double

## 线路终点处的朝向，单位rad

## 请求和响应示例
查询RIL_H 机器人组地图中，LM10848到AP10860的路线信息
## 请求
## {
    "robotGroup": "RIL_H",
    "topoElement": "LM10848-AP10860",
### "elementType": "edge"
## }
## 响应
## {
## "info": {
        "end": 10860,
        "end_dir": -1.5707963267948966,
        "length": 1.7719999999999998,
        "max_speed": -1.0,
        "start": 10848,
        "start_dir": -1.5707963267934772
## }
## }
### 查询 RIL_H机器人组地图中，LM10848点位的信息
## 请求
## {
    "robotGroup": "RIL_H",
    "topoElement": "LM10848",
### "elementType": "point"
## }
## 响应
## {
## "info": {
        "dir": -1.5707963267948966,
        "ignore_dir": false,
## "name": "LM10848"
## }
## }
## 查询不存在的机器人组的场景元素信息
## 请求
## {
    "robotGroup": "SEER GROUP",
    "topoElement": "LM10848",
### "elementType": "point"
## }
## 响应
## {
    "error": "No such group: SEER GROUP"
## }
## 查询不存在的元素信息
## 请求
## {
    "robotGroup": "RIL_H",
    "topoElement": "F",
### "elementType": "point"
## }
## 响应
## {
### "error": "No such point: F"
## }
## 查询不存在的元素类型
## 请求
## {
    "robotGroup": "RIL_H",
    "topoElement": "LM10848",
### "elementType": "region"
## }
## 响应
## {
    "error": "Not supported element type: region"
## }
