# 点位、线路、库位、库区、区块、区域

## 点位、线路、库位、库区、区块、区域
场景中存在用于规划的点位、线路，库位、库区、区域和区块。
点位 和 线路 构成了场景中的拓扑图。调度是在拓扑图中利用 A* 算法规划线路。点位可以有普通点 （LM 点），动作点（AP点）， 充电点 （CP 点），停靠点（PP点）切换地图点（SM 点），线路中可以分为单向路和双向路。
库位 是在点位的基础上增加了动作参数，表示机器人到这个点上做什么动作。动作参数用键值对存储。比如可以存 load 和 unload 动作在库位中。一个点位可以绑定多个库位。一个库位只能绑定一个点位。

库区 是库位的集合。
区块 （Region）是一个矩形块，表示一块空间。区块，点，线路可以形成图形组。图形组可以配置成Mutex 组， MAPF 组， AvoidObsRegion 组等高级组。
一个 区域 通常表示一个楼层的地图。即使不同机器人在不同区域有相同坐标，也不会相撞。不同区域之间可以通过SM 点互联。也就是不同区域见只有SM 点是重名，其他点都不能重名。而 SM 点通常表示实际的电梯位置。
