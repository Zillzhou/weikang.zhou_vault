# 部署调度服务器

## 部署调度服务器
## Windows
如果使用 Mariadb  数据库 安装 Base_RDSInstaller_4.x.x.exe ，如果使用 其他数据库 安装 robod_RobodInstaller_4.x.x.exe
选择安装包后，把安装包拷贝到服务器中默认安装 C:\RDS  非必要不要更改目录
### 安装完毕后 鼠标双击桌面 RDS 快捷方式 运行
## 升级 core 和 rds
## 服务启动 RDS
针对部分用户需要注销服务器账号情况，这时需要使用配置服务来启动 RDS
工具下载地址 https://seer-group.coding.net/api/user/seer-group/project/products/artifacts/repositories/9977006/packages/4337316/versions/13843530/file/download?fileName=WinSystemServer.zip
解压到 RDS/data/WinSystemServer下管理员权限运行 RDS/data/WinSystemServer/autoStart.bat
1661229043784-c9c64e84-ec1a-4d8e-a49a-3cef86db3b74.png

打开服务设置 QtRobodService 属性为自动（任务管理器->服务）
1661229057843-cd337b34-64b3-4545-bbc8-d96d07d87cae.png 

1661229067462-5ec61405-8a5f-4f91-92d2-4b695208b3e7.png

以上步骤做完后重启系统，如果在【任务管理器】未搜索到 RobodPro.exe，请 手动修改 startUp.bat 
把路径改成 RobodPro.exe 所在的 绝对路径，注意用"/" 改完后，再重启系统
## @echo off
start C:/RDS/data/robod/RobodPro.exe
## exit
关闭 Robod 自身开机自启动 (因为 Robod 默认会注册开机自启动，这里的 QtRobodService 也会开机自启动，所以需要关闭)
https://seer-group.feishu.cn/wiki/LPouwQXrSifh45k6tscc9thknQg
