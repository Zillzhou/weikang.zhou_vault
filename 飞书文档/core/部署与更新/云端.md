# 云端

## 云端
Core 在与小车通信过程中，Core 作为客户端，小车作为服务器。Core 部署在云主机上，为了实现 Core 能向小车主动发起网络请求，需要把 Core 加入到小车所在的局域网中，即创建一个虚拟局域网。
## 实现上述内网穿透主要分为两步：
## 1. 搭建虚拟局域网
首先要在云主机上安装虚拟局域网软件，配置 ca 证书等。
然后在网络行为管理网关上，配置上云主机的 IP 和端口、用户名密码等信息，即可成功组网。
### 2. 内网穿透的防火墙规则
组建虚拟局域网后，本地局域网就和云主机处于同一个局域网内。我们再通过 iptables 配置一些防火墙规则，就可以把外网访问重定向到局域网内部。
