双舵轮机器人上顶升电机与舵轮电机选择不同的 CAN 路，可将设备ID改为 1。
需要确定驱动器是否带拨码键盘配置 CAN Node ID 节点。如果有，那么拨码键盘的优先级要高于软件配置，请根据驱动器上的丝印直接通过拨码键盘配置，修改设备站号即修改驱动器的 CAN Node ID，只有拨码键盘无效时软件修改才起效，拨码键盘和拨码对应关系如下图。
