## 步科 7.0 寸 HMI 参数配置教程
### 安装界面编辑软件（ DTools v3.3 及以上）

可在网盘的对应路径下找到此软件的安装包，下载、解压并运行。
按照安装步骤，逐步安装即可。

## 设置 PC 跟触摸屏在同一网段

在 PC 上设置跟触摸屏连接的以太网的 IP 为“192.168.192.x”，x∈[0，20）∪（20,255]。

### 确定触摸屏的 IP 为“192.168.192.20”
注：出厂后的触摸屏 IP 已经修改，可跳过此步骤。如果 PC 连接不上触摸屏，请继续此步骤。

按住触摸屏的屏幕不放，然后给触摸屏上电。
开机后选择【系统设置】，然后选择【简体中文】（或者【English】，根据个人喜好），然后就会跳转到【系统设置】界面。
选择【系统设置】界面上的【IP 设置】标签，如果【IP 地址】不是“192.168.192.20”，则修改【IP 地址】为“192.168.192.20”；如果【网关】不是“192.168.0.1”，则修改【网关】为“192.168.0.1”，其他设置不变。
修改完成之后，点击右上角的【重启】按钮，即可保存当前修改。

### 用 KDManager 工具烧写屏幕界面程序

找到并运行 KDManager.exe 。
点击界面上的【下载处理】，然后点击【设置】，在弹出的【通讯参数设置】界面中设置如下图所示的参数，并点击确定。

在界面上点击如下图所示的【下载工程】按钮，并在弹出的对话框中选择对应的目标界面程序文件（。pkgx），然后就会弹出烧写界面程序的窗口。

烧写界面程序的窗口如下图所示，点击【下载】按钮，就会开始烧写界面程序。
1677203913037-aa386225-17ac-42a1-a718-f0618be8403c.png

烧写成功之后会有如下图所示的提示。
1677203945018-d78df29b-de3e-42b8-956f-4b9889c50118.png

注：当再次烧写时，可能会提示“通讯超时”，如下图所示。是因为界面程序中的程序修改了触摸屏的 IP 地址和网关，需要跳转到【3、确定触摸屏的 IP 为“192.168.192.20”】，重新设置触摸屏的 IP 为“192.168.192.20”。

## 测试验证：
配置完成后，把屏幕 RJ45 通讯网口连接到 SRC 控制器的网口上，启动机器人；
开机后屏幕会显示机器人的 IP 地址，实时电流、电量等信息，如无异常，则配置成功；
