## MOXA网络客户端
此次演示车辆使用外置无线客户端MOXA，配置前应通过网线连接机器人的 Moxa 调试网口，机器人开机后，内置的 Moxa会自动启动，启动时间约为几十秒，听到滴滴两声说明 Moxa 已经启动完成，可以开始使用。
Moxa AWK-1137C 是一款工业无线客户端，可使机器人具备无线功能，稳定的接入无线 Wifi 网络（详情请参考 Moxa 官网 ）。
通过网线连接机器人的 Moxa 调试网口与 PC 的以太网口，参考第二节进行 MoxaAWK-1137C 的配置，即可将机器人通过 MoxaAWK-1137C 接入无线网，此时将上位机接入同一无线局域网，即可通过无线网络连接机器人。
注：下文中提到的“客户端”、“设备”、“Moxa”等均指代“Moxa AWK-1137C”。
## 登录MOXA配置
步骤1：参照上如修改本机PC【以太网】的 IP 地址改为：192.168.127.12（xxx 不能为 253），子网掩码改为：255.255.255.0，默认网关缺省，如下图所示：
1684899974330-de7af6ed-09cb-48d3-b7b5-cc830c701bcc.png

步骤2：打开 Chrome 浏览器输入 http://192.168.127.253 （Moxa AWK-1137C 的默认登录地址)进入 Moxa AWK-1137C 的配置页面，初始用户名：admin，密码：moxa（部分老款MoxaAWK-1137C 产品的初始密码可能为：root），进入moxa配置界面。
1710840220191-970c3d80-090c-4ffd-a88e-ac2e1b874fd3.png 

## 注意：
如果需要修改默认密码，请记住修改后的密码，否则需要重置设备才能恢复（使用回形针戳入设备侧面的 Reset 孔 5 秒），对于已经组装完成的机器人较难操作。
不建议修改 MoxaAWK-1137C 的默认 IP， 禁止 将 MoxaAWK-1137C 设备本身的 IP 获取方式改为 DHCP
当有多台 MoxaAWK-1137C 客户端同时接入同一个网络时可能会出现如下情况：用网线连接到其中一台 MoxaAWK-1137C 客户端，登录 http://192.168.127.253 ，此时网页上显示的设备 MAC 地址与客户端铭牌上的 MAC 地址不相同，这说明登入的后台页面是另一台 MoxaAWK-1137C 的。出现这个情况是因为所有客户端配置后台的默认 IP 地址均为 192.168.127.253，接入同一个局域网后，通过此 IP 地址可能会登入任意一台设备。故建议在安装 Moxa AWK-1137C 时务必记录所安装设备的 MAC 地址，方便在后续调试中使用。该现象的解决方案为使用 Wireless Search Utility 软件，详见下文。
如果网络中已经配置了多台 Moxa AWK-1137C 客户端，此时可使用 Wireless Search Utility 软件查找网络中所有的 Moxa AWK-1137C 设备，并准确进入相应Mac地址的客户端配置页面，操作方法如下：
参考 2.1 的步骤a~c 将 PC 端以太网卡的 IP 地址修改为 192.168.127.xxx 网段并使用网线连接 PC 与 Moxa AWK-1137C
在 PC 上安装 Wireless Search Utility 软件，双击 Wireless Search Utility 图标，在弹出的对话框中直接点击【OK】，如下图所示：

## 设置工作模式
在主界面【Wireless Lan Setup】中点击 【AeroMag】，设置【AeroMag operation mode】为：Inactive，如下图所示：
1684900090434-6fe3fce8-55a6-4ee5-b969-9baeb9b71b94.png

在主界面【Wireless Lan Setup】中选择【Operation Mode】，设置【Wireless】为 Enable，设置【Operation mode】为 Client，如下图所示：
1684900105150-ec916294-c784-4977-bf44-118a2d9c8509.png

## 选择需要连接的网络
在主界面【Wireless Lan Setup】中点击【WLAN】进入【Basic WLAN Setup】界面，点击 【Site Survey】按钮，如下图所示：

1684900121015-962d1c6e-2a8a-4b61-956b-2c8fafa28ce9.png

### 在弹出的网络列表中选择需要连接的网络，如下图所示：
1684900137788-f777ed30-d8e5-4dd6-95eb-87e182c518ad.png

## 注：
如出现网络不可选，请检查天线是否安装正确，如果已安装建议断电重启设备或用回形针长按设备侧面的 Reset 孔 5 秒进行重置。
若需要连接的网络为隐藏网络，请直接在 SSID 选项中输入相应的网络的名称。
### 设置 RFtype 和 Channel width
连接网络完成后，在当前界面根据所要连接网络的路由器属性，正确更改【RF type】和【Channel width】，如下图所示：
1684900159450-ae8da469-c7be-4afe-aea5-a0db7f64dc02.png

### 2.4GHz 网络一般选择 B/G/N Mixed，5Ghz 网络一般选择 A/N Mixed
### c.Channel width 一般选择 20MHz
### d.修改完成后，点击【Submit】按钮进行确认
## 设置网络安全模式和密码
在主界面【Wireless Lan Setup】中点击【WLAN】进入【WLAN Security Settings】页面，在页面中根据所要连接的网络实际情况填写。
【Security mode】：安全模式，目前通常选 WPA2
### 【WPA type】：加密类型，通常选 Personal
【Passphrase】：Wifi 密码（ 请输入正确的 Wifi 密码，否则无法成功连接无线网络 ）
1684900183648-38329427-77eb-4bd6-86e3-02006718630f.png

### 修改完成后，点击【Submit】按钮进行确认
如果网络需要通过企业证书接入，【Security mode】需选择 Enterprise，并在【WLAN Certificate Settings】中导入相应的证书文件，如下图：
1684900197018-6035b74f-0e5e-4a4d-9f34-91d6d448ac6a.png 

1684900209403-8ba18122-0de8-4d2f-8c86-c16aa34f51b2.png 

如果证书文件需要密码，请在【Certificate private password】条目中填写正确的密码。

等待重启完成后，登入设备主页，进入【Status】中【Wireless LAN Status】界面，查看连接信息是否正确，如下图所示为【已连接】状态：

1684900267499-8ca75f72-a0a4-4468-9aa7-a177ac57d354.png

若【Channel】显示为【Not Connected】，则为连接【未连接】状态。如下图所示：

## 确认机器人的 IP 地址

步骤1：在机器人的【高级配置】界面中的【Wifi 客户端配置】标签页，设置 AP 的地址分配模式为 DHCP（我们通常将多个 AP 接入一个路由器，所以需使用路由器的 DHCP 功能，每个 AP 由路由器统一分配 IP），可以查看机器人被分配的无线网络 IP，如下图：
## 无标题.png

去掉【使用 DHCP】的勾选，设置静态 IP、子网掩码、网关信息，填写完成后点击【写入】按钮，等待提示【配置以太网 IP 成功】，然后点击下方的【禁用无线网卡】将机器人的内置无线网卡禁用，避免出现异常冲突。
1710839696412-6f4a9465-db26-49fd-8936-424e09e0da91.png 

步骤2：在【无线网卡配置】栏下，在区域1可通过SSID和密码，将无线网卡接入 Wi-Fi 。点击【高级网络配置】，在区域2输入wifi账号，在区域3配置网络IP。（也可以选择开启DHCP，自动获取IP）
1710900693405-0e64c01b-45f9-48c8-a5e2-23af7107442e.png

区域3可以显示当前连接wifi 。
1710839696412-6f4a9465-db26-49fd-8936-424e09e0da91.png 

步骤3：记住此 IP 后即可断开网线连接，确保 PC 已经接入与机器人相同的无线局域网后，在 Roboshop 首页，点击【输入 IP】按钮，输入机器人的无线 IP 进行连接。
注：如果通过无线 IP 无法连接机器人，请检查 PC 是否正确的接入无线网络、无线路由器的设置是否合理以及是否发生 IP 冲突等情况，如果依然无法连接，请联系我司售后。
