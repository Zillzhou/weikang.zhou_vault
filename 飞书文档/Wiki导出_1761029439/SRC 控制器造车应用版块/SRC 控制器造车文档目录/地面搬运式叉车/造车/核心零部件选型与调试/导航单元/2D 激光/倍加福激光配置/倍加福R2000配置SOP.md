## 配置激光
在链接里下载倍加福激光调试上位机软件，图标所示；
O9iBbSx7TosfKpxgecQcQjHJnib.png

需将个人电脑的以太网网段更改为动态模式(DHCP),将激光雷达上电并通过网线连接个人电脑，打开上位机调试软件，双击“主机PC”，并点击状态栏中的“添加设备”，唤出“R2000 IP COMM”选项框，选中并确定，如图所示；
VKtIbRQn4ohD8XxuQErcmTsYnjd.png

右击左侧的 R2000 IP comm，添加设备，跳出可用的 R2000 型号选项框，选择所需的型号（我司常用 OMDxxx-R2000 HD），如图所示；
L54RboTlloEvv6xTATYcZzYlnSd.png

右击左侧激光(OMDxxx-R2000 HD)图标,在弹出的选项中选择建立连接，然后双击激光(OMDxxx-R2000 HD)图标即可唤出激光配置所有选项，如图所示：
MTQeb54vfo90MWxQK8Nce18Onqd.png

选择 Ethernet Setup 选项，勾选 Enable update of IP configuration 前的方框，然后将 New Settings 中的配置成如图所示：
NcrOb2BJpoBKMrxYinLcl18Nnqg.png

点击工具栏上【写入设备】按钮将新配置的参数上传至倍加福传感器，然后重置传感器【Reboot Device】即可完成配置，如图所示；
HgWWbNbIPoX8X6xHowUcRah4nEd.png

激光设备配置完成后需要对激光设备进行重启，等待重启完成可以使用 CMD 命令行进行 Ping 通测试【ping 192.168.192.100】，测试通过，则说明激光设备 IP 已经配置正确;
## 配置完毕后，需对激光传感器重启
## 配置模型文件
确保激光通讯线与SRC上网口连接，SRC上网口与个人电脑连接，有线网卡网段为192.168.192.x。打开roboshop，进入192.168.192.5机器人。
