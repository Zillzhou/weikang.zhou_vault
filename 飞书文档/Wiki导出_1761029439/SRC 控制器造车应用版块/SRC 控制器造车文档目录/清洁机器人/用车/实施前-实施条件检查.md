## 一、扫图环境要求
机器人对使用的环境是有一定要求的，下面列举了一些常见的不适合机器人运行的环境，请避免这些情况或做相应的处理。
## 1.1 环境的材质
### 以下环境材质会对机器人的建图及定位导航造成影响：
机器人的激光雷达无法正确扫描表面呈镜面反射的物体（如镜子、光滑的金属表面）。
下面两张图为激光雷达的原始数据在普通材质和镜面不锈钢面板的显示。可以看到，普通材质基本呈一个平面，而镜面不锈钢面板数据杂乱，无法看出这是一个平面。

1651914167530-681aa97a-71a8-4ab4-9857-786bb464eaa9.png

机器人的激光雷达无法正确扫描透明后半透明的物体（如玻璃、部分毛玻璃等）。

1651914235923-b00286a6-b7d9-4fcd-87ca-e7185cb7dc4e.png

机器人的激光雷达无法正确扫描黑色的物体，黑色将吸收所有光线，使激光雷达的有效距离降低甚至失效。
如果遇到以上三种情况，需要在环境中的机器人激光雷达所在平面黏贴 3M 反光条，用于遮挡镜面、玻璃或黑色物体。
## 1.2环境的轮廓
### 以下环境会对机器人的建图及定位导航造成影响：
非常空旷的环境，有较多空旷区域的范围大于激光雷达的有效距离，即机器人在这些空旷区域里几乎扫描不到有效的环境特在。
## 解决方法：使用有效距离更大的激光雷达
非常杂乱的环境，机器人无法获得良好的环境轮廓。
## 解决方法：整理环境，移除无用的物体
有较多区域极其相似的环境，机器人无法根据环境轮廓辨别位置。
### 解决方法：在环境中随机增加一些有特征的标志物
没有特征的长廊环境，长廊的两边墙面没有任何可参照物或者参照物非常有规律从而无法获得有效的特征。
### 解决方法：在环境中随机增加一些有特征的标志物
极易变化的环境，短时间内就会发生较大改变的环境将使机器人无法构建出正常的地图，或者使已构建的地图失去可参照性。
## 解决方法：避免或减少环境发生较大的变化
### 二、运行使用环境
除上述环境要求外，应保证地面平坦，无坑洞，若有悬空和边缘的环境，应设置防护栏尽可能远离，另外本公司清洁机器人配备了3D相机坑洞监测、防跌落等功能，可供用户选择。
针对产品在清扫中可能出现的问题，应尽早排除。清理地面上的电源和细小物品避免及机器人受阻、堵塞产品。窗帘、桌布等不要接触地面，防止卷入。
此外，清洁机器人应配备安全合适的充电加水环境，符合本产品对温度、湿度的要求。
在本产品工作范围内应禁止任何易燃易爆物品，远离明火以及禁止堆放可能会导致火灾的物品。严禁使用不符合安全规范和破损的物品（例如插座）。包含本产品在内发生损坏时应立即停止使用，避免危险，应由供应商、厂家等专业人员进行维修或更换。
### 三、网络环境
### 无线网络协议：IEEE 802.11 a/b/g/n
## 宽带速度值：≥100Mbits
## 信号强度要求：≥-60dBm
## 网络延时要求：平均延时时间≤100ms
### 四、Roboshop Pro系统要求
内存最低 4G，如果地图超过 200m*200m，需要 8G+

win7 (x64)、win8 (x64)、win10 (x64)、win11 (x64)
window server 2016 (x64) 、window server 2019 (x64)、window server 2022 (x64)
## Linux
### 支持以下 CPU (x86) 架构的桌面版操作系统
ubuntu18.04、ubuntu20.04、ubuntu22.04、统信uos、银河麒麟
