## 使用中

软件安装—安装软件 USB_CAN Tool （软件及使用手册请联系CAN卡厂商售后）。
硬件连接：准备 USB CAN 卡和一根双绞线，用双绞线的一根连接线将 CAN 卡的 CAN_H 接到 SRC 外接线束 CAN_H 信号上，双绞线的另一根连接线将 CAN 卡的 CAN_L 接到 SRC 外接线束 CAN_L 信号上。如图1 所示：
