## 1. 配置充电桩网络
本文档仅对锂平电池充电桩和叉车锂平电池上 CAN-WIFI 设备模块（致远电子CANWIFI-200T）参数配置进行说明；
无论使用调度系统（SRD、RDS）实现叉车自动充电还是仅使用叉车单车自动充电功能，均需要按照本文档中要求配置充电机上can转wifi模块；

## 上位机配置/测试软件

## 1.2 PC 机与设备网段检测
在使用 PC 机与 CANWIFI-200T 设备通信前， 需要保证用户的 PC 机内有以太网卡。 在使用 PC 机与 CANWIFI-200T 设备通信前，需要知道设备的 IP 地址等网络参数，并且 PC 机与 CANWIFI-200T 设备必须在同一个网段内。
注意：若不知道设备 IP 地址，请恢复设备出厂设置；
DEF 按钮（下图）用于恢复出厂设置，需长按 5-10 秒然后系统 SYS 灯灭，即可放开，设备自动恢复出厂设置，设备在出厂时设定的默认 IP 地址为 192.168.0.178 ,如已知设备 IP ，可跳过此步。
1666320766685-177778bc-9da3-4e09-bb83-943f9c6006b0.png

在知道设备 IP 后，进入 PC 操作系统,然后进入本机的控制面板→进入“网络连接”（WinXP） 或“网络和共享中心”（Win7及以上）→进入“本地连接” 属性→“Internet 协议（TCP/IP）”（WinXP） 或“Internet 协议版本 4（TCP/IP）”（Win7） 属性，修改设备 PC 机 IP 与 CANWIFI-200T 设备在同一个网段内。
1666320766728-1963d8e3-3ae4-4d5d-8c26-234b73d23497.png

至此该 PC 机已可与 CAN-WIFI 设备(CANWIFI-200T)通信并进行配置。
## 1.3 充电桩与机器人接入同一网络SSID
使用网线将设备的 LAN 口连接至 PC 机网口，设备配置时只能用网口配置，保证配置时只连网线。上电充电桩/叉车后打开设备电源开关，启动设备。右击配置软件 ZNetCom Utility，以管理员身份运行软件。
1666320766681-da1ee320-e6db-4e02-abe1-e1088d064d04.png

搜索设备：点击“确认”开始搜索设备。 如果使用搜素设备功能无法搜素设备，选择点击工具栏的【指定搜索】按钮输入 CAN-WIFI 设备 IP 搜索设备。 当前演示设备 IP 为：192.168.0.178。
1666320766669-c23f50f6-e2ed-4df5-9d38-c6b2d92aec6a.png

在搜索窗口中，我们可以看到搜索到的设备，及对应的 MAC 地址和 IP 地址。搜索窗口在 10 秒后自动关闭，用户也可以点击【关闭】按钮让它关闭。
1666320766738-d487f048-528e-4de9-a327-58c9b873284a.png

搜索完成后，被搜索到的设备将出现在 ZNetCom 软件的设备列表中。
1666320767141-6ec21ea3-1860-4194-addd-f34ec0395293.png

双击设备列表中的设备项【获取信息】按钮或属性栏中的【刷新】按钮获取设备信息。

当”获取设备信息”对话框消失以后，用户就可以从左侧属性栏中看到 CANWiFi-200T 设备配置信息。

在左侧属性栏修改设备参数，CAN 配置只修改 CAN0 配置即可（设备支持两个端口，在此电气接线CAN0端口）在此以 192.168.3 网段为例如下图所示

## 注：
1.设备IP属于要连接的WIFI的网段，与机器人、调度系统服务器在同一网络，不可填入 X.X.X.0 或 X.X.X.255， 设备IP 是网络设备（如PC 机、 CANWiFi-200T 等）被指定的一个网络上的地址，在同一网络上它具有唯一性 。
### 2.加密方式设为WPA/WPA2。
### 3.设备IP、子网掩码、网关、DNS、WIFI信息 请根据现场情况填写。
### 4.不填入【当前密码】，更改会失败。
### 5.配置完成后，充电站界面仍显示CAN通信超时，是正常现象。连接成功与否请与调度系统联合实际测试。
1666320767629-9af62769-9437-4841-918e-a761aed8de02.png

### 2. 配置充电桩识别文件
步骤1：【识别文件】>【全部拉取】。
1666320767780-d1eee6e0-52b0-4183-aed3-8fe44fb39bed.png

步骤2：选择【chargespot】，点击【fork】，根据实际配置填写充电桩的【IP】及【port】。
1666320767794-35578a08-d162-40db-a58a-3759433c9c39.png

### 3. 配置模型文件
步骤1：【机器人模型】>【charger】中，将【chargeStartTime】改为【40】s。
1666320767790-40e715c3-fa5f-4489-8477-79217678bf65.png

### 4. 设置CP点
步骤1：根据充电桩的位置，建立CP点，识别模型文件选择【fork.chargerspot】，根据充电桩的实际位置以及充电路线，设置【角度】。并勾选前置点【LM47】，前置点距离约0.5m。
1666320768221-ac3c33d5-615c-412f-83e9-9a748a3c6621.png

步骤2：根据充电桩的实际位置以及充电路线，设置【正走/倒走】，双向路线均要设置。
1666320768412-26cd5134-c469-4382-825a-763df4005c45.png

步骤3：【路径导航】至CP点，开始自动充电。

## 郑重说明
## 1、此次方法仅用于在电池出现馈电情况下的应急处理，必须由经过培训的专业技术人员进行操作。
### 2、此方法仅适用于电池电子原件未损坏、电芯馈电的情况。如果其余故障，请联系厂家售后。
### 3、如果电池馈电过于严重，导致电芯损坏，此方法可能无法充电，此时需要更换电芯。
### 4、电池馈电会对电池容量造成不可挽回的损害，即使此方法恢复电池功能，其容量也可能受损。
