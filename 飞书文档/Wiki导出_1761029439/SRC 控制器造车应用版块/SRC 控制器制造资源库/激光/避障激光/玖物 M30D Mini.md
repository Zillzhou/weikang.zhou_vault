## 配置
(1) 仅支持在 64 位 Windows7 或 Windows10 上存在.NET 4.7.2 运行环境下运行。
(2) 支持我司单线激光雷达通信协议和部分兼容协议。
(3) 需求电脑内存 4G 以上为最佳。
(4) 若雷达通过以太网相连，请关闭 Windows 防火墙。
(5) 若雷达以串口连接线相连，请在 Windows 下安装串口连接线驱动，建议连接线与上位机电脑直连（不通过扩展 坞等转接设备）；
(6) 确保雷达设备已接通电源，设备通过特定连接线连接到计算机或 LAN 端口；
(7) 配置计算机静态 IP 地址，IP 地址为 10.8.8.8，子网掩码为 255.0.0.0，网关为 10.1.1.1（雷达出厂默认网络参数 是 IP:10.8.8.10，子网掩码 255.0.0.0，网关 10.1.1.1，端口号 8089）；
若雷达被修改了网络参数，请保证雷达与 上位机通信时处于同一网段
将激光与笔记本连上之后，将ip和端口按如下配置修改，如果有多个激光，ip和端口需要配置成不一样
