## 一、前言

本说明以 Moxa AWK-1131A 系列为例，其他型号如 AWK-3131A 系列，AWK-4131A 系列等因设置方法基本相同，也可参考本文档。
通常情况下，环境中需要接入多个 AP 以实现 Wi-Fi 信号的覆盖，此时建议将多个 AP 接入一个路由器，由路由器统一管理 AP 及其他客户端的 IP。网络拓扑图如下：
26b51a9e-ec7d-4321-9ca2-94ce440eb3c7-1.png

为实现 Wi-Fi  的无缝漫游功能，通常使用支持 Wi-Fi 漫游的无线客户端（上图中的 Client）接入机器人。

MOXA 客户端的设置请参考该文档：《Moxa AWK-1137C 使用说明》；

### 二、设置调试电脑

设置调试电脑的本地连接的 IP 地址在 192.168.127.150

### 三、设置AP端的参数

### 将测试电脑的网线连接到 AWK-1131A 系列产品上：
439ddde3-53e6-4fcc-ba44-d8330753af0c.png

在产品的电源端子附近有一个 RESET 孔，使用大头针按进去，按住里边的 RESET 按键持续10秒，然后松开，产品重新启动后恢复出厂设置。在浏览器页面输入： 192.168.127.253 （这是MOXA AP 的默认 IP 地址），点击回车进入到 AWK-1131A 的 Web 界面，在弹出的登陆界面中输入用户名：admin，密码：moxa（部分 MOXA 的老款产品密码可能为 root）。输入后登陆产品的参数设置界面，如下图所示：
c446ea35-fb21-43a5-9035-445593166c48.png

首先点击进入 General Setup，设置 AP 的地址分配模式为 DHCP（我们通常将多个 AP 接入一个路由器，所以需使用路由器的 DHCP 功能，每个 AP 由路由器统一分配 IP）：
24273505-9ab3-4c55-87cb-2117a76c796b.png

设置 AP 的射频频段和 SSID，建议使用 5G 频段，即 RF type 为 A:

### 四、进入路由器后台设置

待 AP 重启后，将插在电脑上的网线拔下，将 AP 使用网线接入路由器，登陆路由器后台（不同品牌的路由器后台不尽相同，本文使用 UBNT 路由器作为示例）：
c2ba6881-b784-4f0b-b276-defe672a1820.png

从 AP 的铭牌上可以看到设备的 MAC 地址，如上图为：00:90:E8:66:AA:02。进入路由器后台查找在线客户端，通过 MAC 地址即可找到 AP 以及其被分配得到的 IP 地址：
f01da31d-40d1-4a18-9d2a-143d5f406b61.png

可以看到，图中 AP 的 IP 地址为 192.168.4.120，可以将此 MAC 地址与 IP 绑定住，如下图：
b4461217-d7f8-4612-b201-882790c7a084.png

不同路由器的 IP 与 MAC 绑定方法不同，请以路由器的实际情况为准。
此时，使用测试电脑连接 AP 中设置的 Wi-Fi  SSID (本例中为 Seer_AMB_01) 即可接入网络：
d1cdd074-71c6-4467-bcf2-d98b272f0d24.png

此时，测试电脑是通过 AP 接入网络的，且测试电脑的 IP 地址是由 AP 所连接的路由器分配的，在路由器后台也可以看查看到测试电脑的 IP：
f6db5fb4-4db6-44a7-91ee-d93672c07851.png

IP 地址为 192.168.4.6，也可以设置将其固定。
将测试电脑换成机器人，同理可实现机器人 IP 在路由器中的固定。对于不再使用的机器人，也可以在路由器后台移除固定 IP 的设置。
