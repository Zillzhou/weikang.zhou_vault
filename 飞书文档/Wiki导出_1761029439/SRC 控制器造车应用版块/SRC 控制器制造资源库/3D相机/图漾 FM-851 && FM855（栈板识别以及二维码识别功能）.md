## 设置相机为固定 IP
主控默认有线 IP 为 192.168.192.5 ，因此相机需要修改为 192.168.192.xxx 才可以通过主控正常连接。
使用上位机软件 Percipio Viewer 支持修改网络相机的 IP 地址为静态或动态 IP。设置相机 IP 前，请确保上位机 IP 在目标网段上。
### 按照以下步骤，设置相机 IP：设置相机 ip

 网线连接电脑，打开上位机 percipio， 点击左上角 Preferences，在下拉菜单中点击 Device IP Settings。进入 Device IP Settings 页面。 
ODixbe7KeoDHJuxCNhzcYEXJnTe.png

 在 Net interface list 一栏中选择目标网段的网络接口或者 All network interfaces。 
 在 Found device 一栏中选择目标相机序列号。 
 依次填写相机的 MAC 地址 Device mac address、 目标 IP 地址 Device target IP 、网关 Device target gate、子网掩码 Device target mask。
Device target IP 一栏不填写表示将当前相机设置为动态 IP 地址，填写即表示设置为静态 IP 地址。 
WqxNbRYDFoklXIxLt5ocSgxUnYf.png

### 尤其需要确认子网掩码，必须是 255.255.255.0
点击 Accept。
log 区域提示 “Device IP setting is successful。” 表示设置 IP 成功。 Log 区域提示 “Device IP setting failed” 表示设置 IP 失败，请检查电脑网络配置、硬件接线、 Device IP Settings 页面参数填写是否正确。

图漾相机的线缆定义与颜色并不强相关，需要根据使用的具体型号，使用万用表测量后再接线。
