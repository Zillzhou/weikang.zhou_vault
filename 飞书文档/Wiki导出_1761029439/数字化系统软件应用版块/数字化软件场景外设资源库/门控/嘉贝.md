## 2 、门控盒的输出与输入信号。
## 输入：
## 输入寄存器 0 ：开门到位状态
## 输出：
### 保持寄存器 0 ：开门输出  保持寄存器 1 ：关门输出
Ø  单信号控制开关门：这种模式下，只使用寄存器 0 即可，当值为 1 ：开  门，值为 0 ：关门
Ø  双信号控制开关门：这种模式下，需要使用寄存器 0 、 1 即可，当寄存器 0 的值为 1 ，寄存器 1 的值为 0 时，表示开门，  当寄存器 0 的值为 0 ，寄存器 1 的值为 1 时，表示关门
注意： 门控盒的输出为继电器常开输出，门控盒的输入为 (NPN\PNP) 。
