## 银河麒麟
## 安装方式:
把 Roboshop.deb 拷贝到桌面 （ 注: Roboshop.deb 仅代指安装包 ）
## 安装方式
### 鼠标双击 Roboshop.deb 安装
## 命令行安装:
鼠标在桌面空白处右击显示菜单，选中【打开终端】或【Open in Terminal】
安装命令 sudo dpkg -i ./Desktop/Roboshop.deb （ 注: Roboshop.deb 要改成实际文件名 ） 
安装完成后不要使用 sudo 启动 Roboshop, 否则文件会带锁，鼠标双击图标启动会出现文件没有写入权限问题.
去锁命令: sudo chmod -R 777 /opt/Roboshop/
### 卸载命令: sudo dpkg -r roboshop

## 不支持的系统
## Window 11 N 系统不支持
### 因为该系统限制导致 Roboshop 很多插件无法正常加载
### 所有 CPU 不是 (x86) 架构的桌面版操作系统
## 所有非桌面操作系统均不支持

### Win11 专业版第一次打开慢的处理方法
按 【 Win + X 】 组合键，或 【右键】 点击任务栏上的 【Windows开始徽标】 ，在打开的隐藏菜单项中，选中 【设置】 ；
设置窗口，左侧边栏点击 【隐私和安全性】 ，右侧点击安全性下的 【Windows 安全中心(适用于你的设备的防病毒、浏览器、防火墙和网络保护)】 ；
点击 【打开 Windows 安全中心】 ；
展开左侧的边栏菜单，然后点击 【应用和浏览器控制】 ；
点击 【智能应用控制设置】 ；
 【关闭】 ，这时候系统会提醒“除非重新安装或重置 Windows，否则将无法返回到评估模式”，不用管它，点击 【确认】 就行；
